LLM 总结文献 Prompt

发表于 2024-11-10

总结论文脉络和概述

【放论文pdf】+prompt：请你结合论文内容，用中文分点总结，本文的task（task的输入输出形式），task的挑战及现有方法存在的问题，本文方法解决了什么问题，提出了创新点，有什么contribution和limitation

总结论文方法

【复制论文method部分】+prompt：请你用中文总结本文提出的方法，要求条理清晰，逻辑顺序清楚，能够讲明白本文方法的整个流程，并结合contribution讲明白本文方法的重点，包含所有的公式和变量，不允许遗漏，并解释清楚。行内公式用一个$符号包裹，行间公式用两个$符号包裹。

总结论文实验

【放论文pdf】+prompt：

请你根据本文的experiment部分，

一、总结本文使用的训练集，测试集

二、对本文进行的每个实验，分别用中文总结：

  1. 实验目的：这个实验是为了验证/得出什么结论
  2. 实验使用的测试集
  3. 实验使用的metric，如果是本文新定义的，请你给出公式
  4. 实验比较的方法/baseline，如果是别的工作，请举出3个最重要的比较对象并介绍，如果是和自身的ablation study，请指出对比的baseline的特点，与本文方法比较
  5. 实验的结论

请你完整总结本文提到的每一个实验，不允许遗漏，不要把实验混在一起。行内公式用一个$符号包裹，行间公式用两个$符号包裹。
